    🚀 빠른 시작

    1. 왼쪽에서 템플릿을 선택하여 편집
    2. 🤖 Generate New Template 버튼으로 AI가 템플릿 자동 생성
    3. Test 버튼으로 API 테스트

    📝 템플릿 작성 가이드

    템플릿 이름 규칙

    METHOD-path-to-endpoint.md 형식으로 작성

    • GET-weather.md → GET /weather
    • POST-users.md → POST /users
    • GET-aipm-stories.md → GET /aipm/stories
    • POST-aipm-invest-analysis.md → POST /aipm/invest-analysis

    🔌 API 엔드포인트

    • GET /health - 서버 상태 확인
    • GET /templates - 템플릿 목록
    • GET /template/:name - 템플릿 조회
    • PUT /template/:name - 템플릿 생성/수정
    • DELETE /template/:name - 템플릿 삭제
